- **D. 개인화 2차 — ⏳ 미착수**: 홈 화면 배치 개인화, 관심 업무 핀.

## Stage 1에서 한 것 (완료)

**"각자 보던 화면을 기억해주는 기초 공사."**

- 전에는 업무일정을 "주간"으로 보다가 나갔다 오면 다시 "월간"으로 돌아왔음 (모두 같은 기본 화면).
- 지금은 **누가 로그인했느냐에 따라** 마지막 보던 화면을 기억 — 캘린더 뷰(월/주/목록), 업무현황 탭.
- 핵심: **서버에 저장**되므로 사무실 PC에서 설정한 게 집·폰에서도 그대로 이어짐.
- 이 "개인별 설정 저장소"가 Stage 2·3에서 계속 재사용되는 기초 공사.

## Stage 2에서 할 것 (다음 착수)

1. **내 기준 새 글 배지** (메인): 지금은 "새 글" 배지가 모두에게 똑같이 보임 → 이걸 **내가 마지막으로 확인한 시점 기준**으로 바꿔서, "내가 안 본 글"만 세어주고 페이지에 들어가면 읽음 처리.
2. **필터 기억** (곁들이): 업무현황의 구분·담당자 필터를 걸어두면 나갔다 와도 그대로 걸려 있게. 예: 항상 "담당자: 나"만 보는 사람은 매번 다시 클릭할 필요 없음.

## Stage 3에서 할 것

**업무 카드 순서를 사람마다 다르게.** 지금은 A가 카드 순서를 바꾸면 모두에게 그 순서로 보임(팀 공유) → 각자 자기 순서로 보게 전환. 단, 카드의 상태 이동(진행중→완료)·삭제는 계속 팀 전체 공유. (사용자 확정 사항)

## 권한 Phase 2/3 — 뭘 보류했나

권한 작업을 "뼈대는 지금, 살은 필요할 때"로 나눴고, 그 "살"이 보류분:

- **Phase 2 — 팀원 글쓰기 권한**: 지금 팀원은 보기만 되고 글쓰기(공지 작성·업무 등록·일정 추가)는 관리자만 됨. 팀원도 쓰게 하려면 서버 잠금장치를 풀고 버튼을 열어야 함 → **"팀원이 실제로 글 쓸 일이 생기면 하자"**. ※ 예외: **데모결과 탭은 이미 팀원 쓰기 열려 있음**.
- **Phase 2 — 유관자 장비 제한열람**: 유관자(교원·학생)에게 장비 목록을 보여주되 **가격·제조사는 가린 버전**으로 → "유관자를 실제로 초대할 때 하자". (지금 유관자는 행사·바로가기만 봄)
- **Phase 3 잔여 — 계정 이력장부**: 누가 언제 가입/승인/강퇴됐는지 **기록**. 현재는 "지금 상태"만 보이고 과거 기록 없음 → "추적할 일 생기면 하자".

- **사진 갤러리**: 포털에 팀 사진첩 메뉴 신설. **업로드·열람 모두 팀원**(확정). 썸네일 그리드 → 클릭 시 원본. 무료 용량 내 2~3천 장 수준. "시작해" 하면 바로 제작.
- **데모결과 지표수정방식 논의**: 7/10 집 세션에서 예약한 항목 — 데모결과 탭의 지표값 수정 방식을 **만들기 전에 논의부터** 하기로 함.
