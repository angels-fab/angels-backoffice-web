두 시안 모두 홈 메뉴 없음(로고 클릭 = 홈)이고 실제 메뉴 8개 + 관리자 설정, 실제 배지(빨강 새 글 · 앰버 개선 메모)를 그대로 얹었습니다. 1번은 지금 사이드바처럼 전부 보이는 안, 2번은 메뉴가 늘어날 때를 대비한 그룹 드롭다운 안입니다. 위 폭 토글로 좁은 노트북(1366)과 최악 폭(1024)에서 각 안이 어떻게 버티는지 비교해 보세요. 2번의 그룹 버튼은 실제로 클릭·호버로 열립니다.
