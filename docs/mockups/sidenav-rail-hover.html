1오버레이로 펼친다 (밀어내기와 비교)

펼침 패널이 콘텐츠 위로 떠서, 메뉴를 오갈 때 본문·사진이 흔들리지 않습니다. '밀어내기' 모드를 켜서 화면이 출렁이는 차이를 직접 비교해 보세요.

2닿는 즉시 열린다 (Supabase식)

Supabase 사이드 메뉴처럼 마우스가 스치는 순간 바로 펼쳐지고, 벗어나면 120ms 뒤 닫힙니다. 키보드 Tab 포커스에도 펼쳐지고, 모션 최소화 설정을 존중합니다.

3배지는 아이콘 모서리로

접힘: 빨강 새 글 숫자는 아이콘 우상단, 앰버 개선 메모는 우하단 점. 펼치면 지금처럼 메뉴명 뒤 위첨자 숫자(D7 표준 그대로)로 돌아옵니다.

4터치 화면 유의점

터치 노트북엔 호버가 없어 탭 = 바로 이동(라벨은 못 봄). 내부 도구 메뉴 9개 수준이라 수용 가능하고, 필요하면 나중에 '항상 펼침' 고정 핀을 추가할 수 있습니다.
