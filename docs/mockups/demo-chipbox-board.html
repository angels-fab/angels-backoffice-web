메모 남기기…

1지표 = 사진 바로 아래 칩박스

보드에서 지표를 빼고, 각 제조사 사진 밑에 칩박스 행으로. 소속은 열 위치로 구분(왼쪽 AMAT · 오른쪽 Lam), 그 열의 "+ 지표"는 그 제조사 것.

2칩박스 규격

높이 72px 고정, 폭은 내용 따라 가변(줄바꿈으로 여러 행). 흰색 큰 폰트 지표값+단위가 메인, 조건은 하단 회색 작은 글씨. 지표명은 상단에 아주 작게 — 빼길 원하시면 제거 가능.

3기록 보드 = 제목 없는 메모 전용 · 2열 카드

보드 섹션 제목도, 글마다 붙던 제목도 없음. 메모는 2열 카드로 흘러 빈공간 최소화(작성자·날짜·본문만), 맨 아래 전폭 입력줄. 지표는 칩 "+", 메모는 여기.

4유지되는 것

사진 = 가로스크롤 캐러셀 + 시네마 스크림 + 오버레이 조작부(괜찮다고 하신 그대로). 교차 판정 없음 안내 한 줄. 칩 클릭 = 값 수정(호버 시 연필).
