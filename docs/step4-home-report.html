6) 화면 확인 & 데이터 주의

데이터 주의(투명 공개):
· 캘린더 이벤트에 담당자 필드 없음 → 일정은 시간·유형·장소로 표시
· 업무 상태는 시트에 명시 컬럼 없어 파생(완료=완료일자 / 진행중=share / 예정=시작일 미래 / 지연=그 외 미완료) — 운영 방식과 다르면 조정
· "이번달 도입" = installDate가 이번 달인 장비 수
