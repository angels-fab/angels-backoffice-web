3) Before / After 차이

  • 순흑(#0A0A0C) → 푸른 기 도는 SaaS 다크(#0F1117), 표면 위계 또렷
  • 테두리: 투명 흰선 → 솔리드 슬레이트(#293244)
  • 상태색: 네온기 제거 → 차분한 SaaS 톤
  • 헤더/하단바: 검은 막대 → Surface 톤 + 블러
  • Radius/Focus/Hover를 테마에서 일괄 관리 → 일관성 확보
